Общие правила проекта
Правила упоминаний Правила каналов Правила обращений Остальные правила

2.1.1 Запрещено упоминать роли тех пользователей, которые не смогут решить возникшие проблемы, а также упоминать их без весомых причин Мут 1 день, 3 дня, неделя

2.1.2 Запрещено разбрасываться упоминаниями пользователей без причин или упоминать тех пользователей, которых вы не знаете Мут 1 час, 1 день, бан 3 дня

На верх

2.2.1 Не путайте тему канала Предупреждение, мут 1 час

2.2.2 Каналы обращений не являются чатом для общения, в них мы обсуждаем только по теме обращения Закрытие доступа к обращению + наказание за 2.2.1

2.2.3 Не пишите команды ботов в каналах, которые заняты другими ботами Мут 1 час, 1 день, бан 1 день

На верх

2.3.1 Создавайте только одно обращение с одной темой, не создавайте их по несколько раз, одного обращения вполне достаточно

2.3.2 Если ответа не поступит в течение дня, вы можете пингануть 1 раз администрацию, которая требуется для проверки обращения

2.3.3 Если обращение не принадлежит вам, пожалуйста, не трогайте и не закрывайте его без разрешения

2.3.4 Пожалуйста, после создания обращения, напишите его суть/содержание/причину в течение часа, иначе оно будет закрыто без спроса
